Detailed breakdown — strengths, limitations & fit
Claude
by Anthropic
Best use for
Long document analysis, contract review, complex reasoning, precise Q&A over large procurement files.
Strengths
  • ✓Reads full contracts (100+ pages) in one go
  • ✓Very precise — rarely hallucinates facts or figures
  • ✓Best for complex reasoning and multi-step analysis
Limitations
  • ✗No native Microsoft / SAP integration
  • ✗Separate login — not embedded in daily workflow
Integration & plugins
API available for custom integration. No out-of-the-box Microsoft/Google connector.
Fee
Paid   Free tier limited · Pro from ~$20/month
ChatGPT
by OpenAI
Best use for
Vendor comparisons, structured summaries, general writing tasks, and quick Q&A — great starting point for teams new to AI.
Strengths
  • ✓Most widely known — easiest management buy-in
  • ✓Strong for comparisons, summaries, and drafting
  • ✓Rich plugin / GPT store ecosystem
Limitations
  • ✗Can state incorrect numbers confidently — always verify
  • ✗No native Microsoft/SAP integration without custom build
Integration & plugins
GPT store with 1000+ plugins. Custom GPTs buildable without coding. API available.
Fee
Paid   Free tier available · Plus from ~$20/month · Team plans available
Microsoft Copilot
by Microsoft
Best use for
Teams already on M365 — summarising emails, searching across files, drafting in Word, and meeting recaps in Teams.
Strengths
  • ✓Embedded in Teams, Outlook, Word, Excel — no new tool
  • ✓Searches across your emails, files, and meetings natively
  • ✓Best fit if organisation already uses Microsoft 365
Limitations
  • ✗Requires M365 Copilot licence — significant additional cost
  • ✗Weaker than Claude for deep document analysis
Integration & plugins
Native integration with Teams, Outlook, Word, Excel, SharePoint, and Dynamics 365.
Fee
Paid   ~$30/user/month on top of existing M365 licence

Level 2 · AI Agents
An AI agent receives ONE goal and handles everything — planning, tool use, self-checking, and delivery — without step-by-step instructions from you.
🤖
What makes an AI agent different?
Beyond question & answer

Regular AI = answers your question. AI Agent = receives ONE goal, plans all steps itself, uses tools, checks its own work, delivers a finished result.

1 goal in→Plans steps→Uses tools→Checks work→Result out

Contract matching agent
Which contract covers this PR?
Reads every incoming PR, searches all active contracts, instantly identifies the right match — with spend headroom check and expiry warning.

Output: KTR-2024-0088 — welding materials, valid Dec 2025, IDR 180M headroom. PR (IDR 22M) within limit. Alt: KTR-2024-0045 expires in 45 days — not recommended.

🔗
PR bundling agent
Group PRs for maximum savings
Scans all open PRs, clusters by category and timing, estimates savings from bundling, and flags exceptions that should not be grouped.

Output: Bundle A: 7 MRO PRs (IDR 340M) — est. 12-15% savings. Bundle B: 4 IT peripheral PRs (IDR 89M). ⚠️ PR-0418 (lab chemicals): do NOT bundle.

Contract expiry monitor
Zero contracts lapse unnoticed
Scans all active contracts daily, cross-checks which open PRs depend on each, and sends proactive alerts before it is too late to act.

Output: 🔴 KTR-2024-0033 expires April 20. 6 open PRs depend on this. 🟡 KTR-2024-0051 expires May 15. No renewal started.

Price benchmarking agent
Is this vendor quote fair?
Compares vendor quote against your historical spend data and market rates. Generates a specific counteroffer with reasoning.
